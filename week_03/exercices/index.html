Exercises

The output of all the exercises in this html are going to be displayed in the console.

Arrays

Insert the array elements separated by semicolons i.e "num_1;...;num_j;...;num_N", do not put any quotation mark. If another syntax for the array is introduced there is no guarantee for this exercise to work well.
If no array string is specified it will use the array [1,7,0,34,23,5,2] as default.

Functions

Insert the array elements separated by semicolons i.e "item_1;...;item_j;...;item_N". If another syntax for the array is introduced there is no guarantee for this exercise to work well.
If no array string is specified it will use the array ['school', 'js', 'the', 'is', 'this'] as default.

Time

Insert the time in seconds to start the Countdown.
If the number is not specified it will start from 300 seconds (5 minutes).

Storage

Insert the note you want to store.
If the number is not specified it will save "JS Rules."
